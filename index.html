Tentang Saya

Saat ini saya menempuh studi Magister Teknik Informatika di Universitas Hasanuddin, di mana saya mendalami riset pada bidang sistem pendukung keputusan dan web development modern. Latar belakang pendidikan ini memberikan saya pemahaman kuat tentang dasar-dasar ilmiah di balik teknologi.

Saya aktif mengeksplorasi teknologi melalui project hands-on dan sertifikasi profesional. Setiap project adalah kesempatan untuk menerapkan teori ke dalam praktik dan belajar dari setiap iterasi. Saya percaya bahwa kolaborasi dan feedback konstruktif adalah bagian penting dari proses pertumbuhan.

Fokus saya saat ini adalah membangun aplikasi web yang responsif, efisien, dan user-friendly, sambil terus mengembangkan kemampuan dalam mengimplementasikan algoritma SPK untuk solusi bisnis.
